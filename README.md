# spring-jdbc-demo
Demo project for Spring boot starter jdbc (Spring boot 2.4.0)

- SpringJdbc常用功能示例
- 使用[COLA应用架构](https://github.com/alibaba/COLA)
- 使用`NamedParameterJdbcTemplate`与`FilterCondition`扩展SpringJdbc功能，灵活构造SQL语句
